Comment lancer la correction : 
	1) Ouvrir un terminal dans le dossier meteor du bureau
	2) Créer un nouveau projet avec "meteor create nom_du_projet"
	3) Se déplacer dans le projet avec "cd nom_du_projet"
	4) Supprimer les dossiers Client et Server du nouveau projet
	5) Copier / coller TOUT le dossier et les fichiers de la correction dans le nouveau projet
	6) Taper "chmod 755 to_execute"
	7) Taper "./to_execute"
	8) Attendre l'installation des modules
	9) Lancer le serveur avec "meteor"
